# 归墟灯录

原创武侠志怪探案角色卡前端原型。

## 文件

- `index.html`：网页原型主文件，可直接用浏览器打开。
- `assets/gui_xu_dossier_bg.png`：原创视觉底图资产。

## GitHub Pages

1. 在 GitHub 新建仓库，例如 `guixu-denglu`。
2. 上传本文件夹内的所有文件和 `assets` 文件夹。
3. 进入仓库 `Settings -> Pages`。
4. Source 选择 `Deploy from a branch`，Branch 选择 `main`，目录选择 `/root`。
5. 等待部署完成后访问 GitHub Pages 地址。
